EduPrompt Studio

Tools edukasi untuk membantu guru, kreator pembelajaran, dan orang tua menghasilkan prompt visual pembelajaran yang terstruktur, konsisten, dan sesuai jenjang PAUD, SD, dan SMP.

Apa yang Bisa Dibantu Tools Ini?

  • Menyusun prompt visual untuk worksheet, workbook, infografis, dan LKPD
  • Menyesuaikan visual dengan jenjang PAUD, SD, dan SMP
  • Membantu struktur konten sesuai Kurikulum Merdeka
  • Mengurangi trial-error saat membuat visual pembelajaran

Digunakan Untuk Apa Saja?

  • Worksheet: Visual aktivitas belajar seperti mencocokkan, memberi label, tracing, dan soal sederhana.
  • Workbook: Materi bertahap dengan kombinasi penjelasan singkat dan latihan siswa.
  • Infografis: Storytelling visual untuk menjelaskan konsep pembelajaran secara ringkas.
  • LKPD: Lembar kerja siswa berbasis aktivitas dan refleksi.

Contoh Visual Pembelajaran

Contoh berikut menunjukkan fleksibilitas visual yang dapat digunakan untuk berbagai mode pembelajaran dan jenjang.

Fleksibel untuk Berbagai Mode & Gaya Visual

  • Mode: Worksheet, Workbook, Infografis, dan LKPD
  • Jenjang: PAUD, SD, dan SMP
  • Gaya visual dapat disesuaikan dengan kebutuhan pembelajaran

Panduan Menggunakan EduPrompt Studio

  1. Tentukan jenis konten, topik, dan jenjang siswa
  2. Gunakan EduPrompt Studio untuk menyusun prompt visual
  3. Salin prompt yang dihasilkan
  4. Tempelkan ke image generator seperti Nano Banana
  5. Gunakan visual untuk materi pembelajaran
